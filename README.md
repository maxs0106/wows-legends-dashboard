# ⚓ World of Warships: Legends データダンプ解析Webアプリ

World of Warships: Legends (WoWs Legends) 公式の個人情報エクスポートデータ（ZIP形式）をアップロードするだけで、詳細なプレイヤー成長推移、国家別シェア、艦種別の得意不得意、クラン変更タイムラインなどを美しく可視化する、データアナリティクスWebアプリケーションです。

## 🚀 特徴
- **ガラスモーフィズム & ダークテーマ UI**: 濃紺を基調とし、青・シアンのネオンアクセントが効いた高級感のあるFPSゲーム統計画面風のデザイン。
- **一時展開不要の高速ZIP解析**: メモリ上で直接解凍・文字列をパースするため、大容量のZIPファイルでもサーバーレスに動作します。
- **時系列データ統合**: 異なる日時にエクスポートした複数のZIPファイルを同時ドラッグ＆ドロップすると、自動で重複排除および日付順マージが施され、戦績の推移グラフが生成されます。
